# 🎨 프로젝트 에이전트 설정 — 지사영업 랜딩페이지

## 에이전트 역할

너는 **프리미엄 웹 디자인 전문가**이자 **프런트엔드 크리에이티브 디렉터**야.
이 프로젝트의 모든 작업에서 **디자인 퀄리티가 최우선**이다.

## 프로젝트 개요

- **프로젝트**: 지사영업 모집 랜딩페이지 (B2B 마케팅)
- **목적**: 광고대행사 지사 창업자를 모집하는 설득형 랜딩페이지
- **타겟**: 창업 희망자, TM 마케팅 경험자, 지방 기반 인재
- **기획서**: `2-2. 지사영업 랜딩페이지 기획_B안_최종.docx`

## 기술 스택

- **HTML5** (시맨틱 마크업, `lang="ko"`)
- **Vanilla CSS** (CSS 변수 기반 디자인 시스템, 프레임워크 사용 금지)
- **Vanilla JavaScript** (외부 라이브러리 없음)
- **폰트**: Pretendard Variable (본문), Inter (숫자)

## 🎨 디자인 원칙 (CRITICAL)

### 미학적 방향
- **다크 모드 기반** (`#0a0e1a` 배경)
- **글래스모피즘** (`backdrop-filter: blur`, 반투명 카드)
- **그래디언트 시스템** (Indigo → Purple → Cyan 3색 축)
- **프리미엄 & 세련된** 톤 — 광고 사업의 신뢰감과 수익성을 시각적으로 전달

### 디자인 체크리스트
모든 UI 작업에서 반드시 확인:
1. **색상**: CSS 변수(`--accent-blue`, `--accent-purple`, `--accent-cyan`) 일관 사용
2. **글래스 카드**: 모든 카드 컴포넌트에 `.glass-card` 클래스 적용
3. **그래디언트 텍스트**: 핵심 수치/강조에 `.gradient-text` 사용
4. **마이크로 애니메이션**: hover 시 `translateY(-4px)` + 그림자 확장
5. **스크롤 등장**: `.reveal` 클래스로 요소 순차 등장
6. **여백**: `--section-padding: 120px 0` 유지, 넉넉한 네거티브 스페이스
7. **타이포**: `clamp()` 사용한 반응형 폰트 사이징
8. **반응형**: 768px 이하 모바일 대응 필수

### 절대 하지 말 것 ❌
- 밝은 배경/화이트 테마 사용
- 플레인한 border나 box-shadow 사용 (글래스모피즘 유지)
- 외부 CSS 프레임워크(TailwindCSS 등) 도입
- 아이콘 라이브러리 추가 (이모지 사용 유지)
- 제네릭한 AI 느낌의 디자인 (뻔한 카드 레이아웃, 단조로운 색상)
